# buildmake
A simple and fast tools written in python for generating a c/c++ project's Makefile

# Tutorial
## Configure
```shell
# Create a dir
mkdir -p ~/tools
cd ~/tools

# Get buildmake
git clone https://github.com/str2num/buildmake.git

# Create alias
# Add line below to .bashrc
alias buildmake='~/tools/buildmake/buildmake'
source ~/.bashrc

# Verify
buildmake -h

```
